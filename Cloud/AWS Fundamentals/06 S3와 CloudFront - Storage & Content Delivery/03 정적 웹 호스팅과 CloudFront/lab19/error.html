요청을 처리할 수 없습니다

이 파일은 Amazon S3 정적 웹 호스팅의 Error document로 쓰는 예제입니다.

오류가 난 요청에 대해 S3는 HTTP 리다이렉트(다른 URL로 보내기)가 아니라, 지정해 둔 이 HTML을 응답 본문으로 돌려주는 방식으로 동작합니다. 브라우저 주소창에는 원래 요청한 경로가 남을 수 있습니다.

표시되는 오류의 종류는 설정·경로에 따라 달라지므로, 이 페이지는 특정 상태 코드(예: 404)에만 묶지 않고 에러 응답용 통합 문서로 두었습니다.
