# 03 셸과 CLI / 02 파일과 디렉토리

경로 체계를 이해하고, 파일과 디렉토리를 탐색·생성·복사·이동·삭제하는 기본 명령을 다룬다.

## What you will learn

- 절대 경로와 상대 경로, 특수 경로 표기 (., .., ~, -)
- pwd, cd, ls 탐색 명령과 ls -l 파일 유형 구분 (-, d, l, c, b)
- touch, mkdir (-p), cp (-r), mv, rm (-r, -f) 파일/디렉토리 조작
- 와일드카드(Globbing): *, ?, [], {} — 셸이 명령 실행 전에 확장한다
