# 4.7 Comprehension

## 개요

**Comprehension(컴프리헨션)**은 **이터러블을 순회하면서 새 리스트·집합·딕셔너리**를 한 줄로 만드는 문법이다. `for` 루프로 append하거나 반복문을 여러 줄 쓰지 않고, **`[표현식 for ...]`**, **`{표현식 for ...}`**, **`{키: 값 for ...}`** 형태로 list, set, dict를 구성할 수 있다. 조건(`if`)을 넣어서 일부만 포함시키거나, 중첩 루프로 조합을 만들 수도 있다. 개념부터 **list**, **set**, **dict** comprehension까지 다양한 예제로 익히는 것이 이 단원의 목표다.

---

## 핵심 개념

### Comprehension이란

Comprehension은 **"반복 가능한 것에서 새 컬렉션을 만드는"** 한 줄 식이다. 동일한 동작을 `for` 루프로 쓰면 여러 줄이 되는데, 이를 **대괄호·중괄호** 안에 `for ... in ...` 형태로 압축한 것이다. 결과 타입에 따라 **list comprehension** `[...]`, **set comprehension** `{...}`(값만), **dict comprehension** `{k: v for ...}`로 구분한다. 제너레이터 표현식 `(...)`은 이 단원 범위에서 다루지 않고, list/set/dict 세 가지에 집중한다. 공통적으로 **이터러블을 한 번 순회**하며, 각 요소(또는 조합)에 대해 **표현식**을 계산해 새 컬렉션의 요소(또는 키-값)로 넣는다.

### List comprehension

**List comprehension**은 `[표현식 for 항목 in 이터러블]` 형태다. 이터러블의 각 항목에 대해 표현식을 계산한 결과를 **순서대로** 리스트에 넣는다. `if 조건`을 끝에 붙이면 조건을 만족하는 항목만 포함된다. `for`를 여러 개 쓰면 **중첩**이 되어, 앞의 이터러블과 뒤의 이터러블의 **조합**처럼 동작한다. 왼쪽에서 오른쪽 순서로 for가 진행되므로, "바깥 루프 → 안쪽 루프" 순서와 같다.

```python
# 기본: 각 요소에 1을 더한 리스트
a = [x + 1 for x in [1, 2, 3]]
print(a)   # [2, 3, 4]

# if 조건: 짝수만
b = [x for x in range(6) if x % 2 == 0]
print(b)   # [0, 2, 4]

# 중첩: (i, j) 조합을 (i, j) 튜플 리스트로
c = [(i, j) for i in [1, 2] for j in ["a", "b"]]
print(c)   # [(1, 'a'), (1, 'b'), (2, 'a'), (2, 'b')]
```

### Set comprehension

**Set comprehension**은 `{표현식 for 항목 in 이터러블}` 형태다. list와 문법이 비슷하지만 **중괄호**를 쓰고, 결과가 **set**이므로 **중복이 제거**되고 **순서가 보장되지 않는다**. 리스트에서 중복을 제거하면서 변환할 때 유용하다. `{x % 3 for x in [1, 2, 3, 4, 5]}`처럼 "나머지 값들의 집합"을 만들 수 있다.

```python
# 제곱의 집합 (중복 없음)
s = {x ** 2 for x in [1, -1, 2, -2, 3]}
print(s)   # {1, 4, 9}

# 문자열 길이 집합
lens = {len(w) for w in ["a", "ab", "abc", "a"]}
print(lens)   # {1, 2, 3}
```

### Dict comprehension

**Dict comprehension**은 `{키표현식: 값표현식 for 항목 in 이터러블}` 형태다. 각 항목에 대해 **키**와 **값**을 계산해 한 쌍으로 넣는다. 키가 중복되면 **나중 값**이 남는다. 리스트를 "키 → 값" 매핑으로 바꾸거나, 두 리스트를 zip으로 묶어 dict를 만들 때 자주 쓴다.

```python
# 숫자 → 제곱
squares = {x: x ** 2 for x in range(1, 5)}
print(squares)   # {1: 1, 2: 4, 3: 9, 4: 16}

# if 조건: 짝수만 키로
evens = {x: x * 2 for x in range(5) if x % 2 == 0}
print(evens)   # {0: 0, 2: 4, 4: 8}
```

### if 조건·중첩 for

**`if`**는 `for ... in ...` 뒤에 붙인다. `[x for x in it if cond]`는 조건을 만족하는 x만 표현식에 넣는다. **여러 `for`**는 왼쪽이 바깥, 오른쪽이 안쪽이다. `[(i, j) for i in A for j in B]`는 A의 각 i에 대해 B의 각 j를 짝지어 (i, j) 리스트를 만든다. **if를 여러 개** 쓸 수도 있다. `[x for x in it if c1 if c2]`는 c1과 c2를 모두 만족할 때만 포함된다(실무에서는 `if c1 and c2` 한 개로 쓰는 경우가 많다).

```python
# if로 필터
lst = [x for x in range(10) if x % 2 == 1]
print(lst)   # [1, 3, 5, 7, 9]

# 문자열 리스트 → 길이 2 초과만 대문자로
words = ["a", "ab", "abc"]
upper_long = [w.upper() for w in words if len(w) > 2]
print(upper_long)   # ['ABC']
```

### [실습] lab47

이 실습에서는 **list, set, dict comprehension**의 개념을 다양한 예제로 복습한다.

#### ① List comprehension — 기본·if

`script01.py`

```python
# 1~5의 두 배
doubled = [x * 2 for x in range(1, 6)]
print(doubled)

# 10 미만 중 3의 배수
threes = [x for x in range(10) if x % 3 == 0]
print(threes)
```

실행 결과: `[2, 4, 6, 8, 10]` / `[0, 3, 6, 9]`

#### ② List comprehension — 중첩 for

`script02.py`

```python
# 2x2 조합
pairs = [(i, j) for i in [1, 2] for j in [10, 20]]
print(pairs)

# 문자열 리스트의 각 문자를 펼쳐서 리스트
chars = [c for s in ["ab", "cd"] for c in s]
print(chars)
```

실행 결과: `[(1, 10), (1, 20), (2, 10), (2, 20)]` / `['a', 'b', 'c', 'd']`

#### ③ Set comprehension

`script03.py`

```python
# 나머지 값 집합 (중복 제거)
mods = {x % 4 for x in range(10)}
print(mods)

# 단어 길이 집합
words = ["hi", "hello", "hi"]
lens = {len(w) for w in words}
print(lens)
```

실행 결과: `{0, 1, 2, 3}` / `{2, 5}`

#### ④ Dict comprehension

`script04.py`

```python
# 인덱스 → 값
lst = ["a", "b", "c"]
d = {i: v for i, v in enumerate(lst)}
print(d)

# 키-값 뒤집기 (값 → 키, 단 값이 유일할 때)
original = {"a": 1, "b": 2, "c": 3}
reversed_d = {v: k for k, v in original.items()}
print(reversed_d)
```

실행 결과: `{0: 'a', 1: 'b', 2: 'c'}` / `{1: 'a', 2: 'b', 3: 'c'}`

#### ⑤ List + if — 문자열·타입 활용

`script05.py`

```python
# 문자열 리스트에서 길이 2 이상만
words = ["I", "am", "Python"]
long = [w for w in words if len(w) >= 2]
print(long)

# 숫자 리스트에서 양수만 제곱
nums = [1, -2, 3, -4, 5]
squares_pos = [x ** 2 for x in nums if x > 0]
print(squares_pos)
```

실행 결과: `['am', 'Python']` / `[1, 9, 25]`

#### ⑥ 실습 포인트

- **List**: `[expr for x in it]`, `[expr for x in it if cond]`, `[expr for a in A for b in B]`.
- **Set**: `{expr for x in it}` — 중복 제거, 순서 미보장.
- **Dict**: `{k: v for x in it}` 또는 `{k: v for k, v in it.items()}`. 키 중복 시 나중 값 유지.
- **if**는 `for` 뒤에. **중첩 for**는 왼쪽이 바깥 루프.

## 주의사항 (Pitfalls)

- **가독성**: 한 줄이 너무 길어지면 여러 줄로 나누거나, 루프로 풀어 쓰는 편이 낫다. PEP 8에 맞게 줄바꿈할 때는 대괄호·중괄호 안에서 연속된 줄은 들여쓰기로 구분한다.
- **dict comprehension과 `{}`**: `{x for x in ...}`는 set, `{x: y for ...}`는 dict다. `{}`만 쓰면 dict이므로, 빈 set은 `set()`으로 만든다.
- **중첩 순서**: `[f(i,j) for i in A for j in B]`는 A의 각 i에 대해 B 전부 순회한 뒤 다음 i로 간다. 순서를 바꾸면 결과 리스트의 순서도 바뀐다.
- **부수 효과**: comprehension 안에서는 표현식만 쓰는 것이 좋다. append 같은 부수 효과를 넣지 말고, "새 컬렉션을 만드는 것"에만 쓰면 예측 가능하다.

## Deep Insight

Comprehension은 **선언적**에 가깝다. "어떤 조건의 어떤 변환으로 컬렉션을 만든다"를 한 식에 적을 수 있어, 반복문과 append를 여러 줄 쓰는 것보다 의도가 드러나기 쉽다. list/set/dict가 같은 "for ... in ..." 구조를 공유하므로, 한 종류를 익히면 나머지도 문법만 바꿔 적용할 수 있다. 다만 복잡한 중첩이나 여러 if가 붙으면 가독성이 떨어질 수 있으니, 그때는 일반 for 루프로 풀어 쓰는 것이 유지보수에 유리하다.

## 요약

- **Comprehension**: 이터러블을 순회하며 **새 list/set/dict**를 한 줄로 만드는 문법.
- **List**: `[expr for x in it]`, `if` 조건, `for a in A for b in B` 중첩.
- **Set**: `{expr for x in it}` — 중복 제거. **Dict**: `{k: v for ...}` — 키-값 쌍 구성.
