# 01 운영체제의 이해 / 05 UNIX의 역사와 설계 철학

UNIX의 탄생(1969)부터 현재까지, 기술적 역사와 산업계의 흐름을 따라가며 현대 OS들의 뿌리를 찾아본다. BSD, System V, Windows NT, macOS, GNU/Linux가 어디서 갈라지고 어떻게 발전해왔는지를 다룬다.

## What you will learn

- UNIX의 탄생과 C 언어에 의한 이식성 혁명 (1969-1973)
- UNIX 설계 철학의 네 가지 원칙과 앞 섹션에서 체험한 것과의 연결
- BSD가 TCP/IP를 구현하여 인터넷의 기반을 만든 과정
- System V와 UNIX 전쟁이 시장을 파편화시킨 역사
- 상용 UNIX 벤더들(Sun, IBM, HP, DEC, SGI)의 전성기와 쇠퇴
- Windows NT가 UNIX가 아닌 VMS 갈래라는 사실
- macOS/iOS가 NeXTSTEP(Mach + BSD)에서 출발한 공인 UNIX이라는 사실
- GNU 프로젝트의 "커널 없는 OS"와 Linux가 그 공백을 채운 과정
- 현대 OS 전체의 계보도
