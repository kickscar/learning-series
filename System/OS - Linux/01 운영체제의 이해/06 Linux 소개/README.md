# 01 운영체제의 이해 / 06 Linux 소개

21세의 학생이 만든 "취미 프로젝트"가 어떻게 서버, 클라우드, 모바일, 슈퍼컴퓨터까지 지배하는 산업의 기반이 되었는가. Linux 커널의 탄생부터 배포판 생태계, 엔터프라이즈 도입, 현재 위치까지를 다룬다.

## What you will learn

- Tanenbaum-Torvalds 논쟁(1992)과 모놀리식 커널의 실용적 성공
- 최초의 배포판들(MCC, SLS, Slackware, Debian)과 배포판 생태계의 형성
- Red Hat의 "소프트웨어는 무료, 서비스는 유료" 비즈니스 모델
- CentOS 종료(2020)와 Rocky Linux 탄생의 맥락
- 커널 개발 모델과 Git의 탄생(2005)
- IBM $10억 투자(2000), SCO 소송 패소(2011), 증권거래소의 Linux 이전
- LAMP 스택, 클라우드(AWS, GCP, Azure), Android, 슈퍼컴퓨터 100%
- Linux의 현재 위치: 서버 70%+, 클라우드 90%+, 모바일 72%
