# 03 가상 네트워크 / 01 가상 네트워크 유형

VirtualBox의 5종 네트워크 모드(NAT, NAT Network, Bridged, Internal, Host-Only)의 차이를 이해하고, 직접 구성하여 통신 범위를 확인한다.

## What you will learn

- 5종 네트워크 모드의 구조와 통신 범위 차이
- Internal Network가 서브넷 격리의 핵심인 이유
- Host-Only로 호스트에서 VM에 관리 접속하는 방법
- 하나의 VM에 여러 NIC를 조합하여 복합 네트워크 구성하는 방법

## Examples

- `lab01` → NAT 외부 통신, Internal VM 간 통신, Host-Only 호스트 통신 확인
