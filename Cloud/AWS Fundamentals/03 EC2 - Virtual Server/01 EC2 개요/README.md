# 03 EC2 - Virtual Server / 01 EC2 개요

EC2(Elastic Compute Cloud)의 기본 개념을 정리한다. EC2 Instance가 AMI/Instance Type/Network(ENI)/Security Group/EBS/Key Pair 조합으로 만들어진다는 관점에서, 다음 Section의 Instance 생성과 SSH 접속 실습을 위한 기준을 잡는다.

## What you will learn

- EC2 서비스와 EC2 Instance의 차이
- EC2 Instance를 구성하는 핵심 요소(AMI/Type/ENI/SG/EBS/Key Pair)
- Instance Type 네이밍과 Free Tier/비용 감각
- AMI 선택 기준과 Public/Private AMI 구분
- Key Pair(SSH)와 IAM Credential의 역할 차이
