# 01 Cloud와 AWS 시작하기 / 03 AWS 계정과 Console

AWS 계정 생성 절차의 핵심 포인트를 정리하고, AWS Management Console에서 Region/Identity/서비스 탐색의 기준을 잡는다. Billing Dashboard와 Budget Alert까지 확인해 실습 전 비용 통제 루틴을 만든다.

## What you will learn

- AWS 계정(Account)과 Root User의 의미, 보안 주의사항
- AWS Management Console의 기본 UI 요소(서비스 검색, Region, 계정 메뉴)
- 실습 관점에서 Region을 고정해야 하는 이유와 실수 패턴
- Free Tier/Billing Dashboard 확인과 Budget Alert 설정의 목적

## Examples

- `lab01` → 계정 생성, Console 로그인, Region 고정, 서비스 탐색, Billing Dashboard 접근 경로 확보
- `lab02` → AWS Budget Alert 설정(Budgets 생성, Email 알림 설정)
