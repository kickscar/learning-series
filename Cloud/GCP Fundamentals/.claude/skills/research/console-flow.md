# Skill: research/console-flow

## 목적

콘솔의 메뉴 경로, 페이지 뎁스, 영문/한글 용어를 조사하고
writer가 실습 단계를 정확하게 작성할 수 있도록 reference를 제공한다.
범용 설계 — 어떤 서비스 섹션에도 적용 가능.

> 콘솔 이름, 경로 형식, 콘솔 언어 기준은 `CLAUDE.md > 플랫폼 상수` 참조.

---

## 조사 항목

### 1. 콘솔 메뉴 경로
해당 섹션에서 사용하는 서비스의 콘솔 진입 경로.
콘솔 이름은 `CLAUDE.md > 플랫폼 상수` 기준.

```
예시)
{콘솔} 홈 → [검색] {서비스명} → Create → {리소스 유형}
```

- 각 클릭 단계를 순서대로 기록
- 탭, 블레이드, 설정 패널 뎁스 포함

### 2. 용어 매핑 테이블
실습에 등장하는 모든 UI 용어를 영문/한글 매핑으로 정리.

| 영문 (기준) | 한글 콘솔 | 비고 |
|------------|----------|------|
| Resource group | 리소스 그룹 | |
| Virtual machine | 가상 머신 | |
| ... | ... | |

- **영문이 기준**, 한글은 참고용
- 한글 콘솔 용어가 어색하거나 불일치 시 비고에 표시

### 3. 주요 설정 필드
실습에서 직접 입력하거나 선택하는 필드명 목록.

---

## 출력 형식

`.claude/research/{ch_no}.{sec_no}.md` 파일로 저장.

```markdown
# Research: {ch_no}.{sec_no} — {섹션 제목}

> 조사일: YYYY-MM-DD
> 기준: {콘솔 이름} (CLAUDE.md > 플랫폼 상수 참조)

## 콘솔 경로

### {실습명}
1. Google Cloud Console 홈
2. 검색창 → "{서비스명}"
3. ...

## 용어 매핑

| 영문 (기준) | 한글 콘솔 | 비고 |
|------------|----------|------|
| ... | ... | |

## 주요 설정 필드

- {필드명}: {설명}

## 조사 메모

{버전 변경, UI 변경 이력, 주의사항 등}
```

---

## 조사 방법

> 공식 문서 URL과 콘솔 이름은 `CLAUDE.md > 플랫폼 상수` 참조.

1. 공식 문서 검색
2. 콘솔 실제 UI 경로 확인 (최신 스크린샷 문서 기준)
3. 한글 콘솔 용어는 공식 한국어 문서 기준으로 확인
