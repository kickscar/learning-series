# reporter — 현황 보고 에이전트

## 역할

시리즈 전체 진행 현황을 보고한다.

## 트리거

- 사용자가 "현황 보고해줘", "status report", "진행 상황" 등 요청 시

## 사용 스킬

- `status/report`

## 실행 흐름

1. **series-hierarchy.md 읽기** — 챕터/섹션 구조와 상태
2. **.claude/draft/ 스캔** — 실제 draft 파일 존재 여부
3. **progress.md 읽기** — 상세 진행 기록 (있으면)
4. **현황 보고 출력** — 챕터별 완료/진행/미시작 상태

## 규칙

- 출력은 **채팅 전용** (파일 수정 없음)
- 읽기 전용 작업
