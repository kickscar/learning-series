# Built-in Scalar Types
이 디렉터리의 예제들은 Python의 **내장 스칼라 타입** `int`, `float`, `str`, `bytes`, `bool`의 기본 동작을 이해하기 위해 구성되어 있다.
각 타입의 리터럴 표현, 기본 연산, 그리고 실무에서 자주 발생하는 주의사항을 코드로 직접 확인하는 것이 목적이다.

## What These Examples Explain
이 예제들은 다음 개념을 설명한다.

- 정수(`int`)의 임의 정밀도와 산술 연산
- 실수(`float`)의 부동소수 오차와 비교 방법
- 문자열(`str`) 연산과 f-string 포맷
- `str`과 `bytes`의 차이 및 인코딩/디코딩
- `bool` 타입과 진릿값(truth value) 규칙

## Examples
- `lab01` → 정수 리터럴과 기본 연산
- `lab02` → 부동소수 오차와 `math.isclose`
- `lab03` → `encode` / `decode`
- `lab04` → bool과 조건문 문맥
- `lab05` → 문자열 연산과 f-string
