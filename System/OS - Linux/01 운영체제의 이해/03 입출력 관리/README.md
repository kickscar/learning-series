# 01 운영체제의 이해 / 03 입출력 관리

프로그램의 입출력 코드가 실행될 때 커널 내부에서 어떤 경로를 거쳐 하드웨어 장치에 도달하는지를 다룬다. 시스템 콜과 디바이스 드라이버의 역할을 이해한다.

## What you will learn

- 프로그램 → 라이브러리 → 시스템 콜 → 커널 → 디바이스 드라이버 → 하드웨어의 I/O 경로
- 시스템 콜이 I/O에서 구체적으로 어떻게 동작하는가
- 디바이스 드라이버가 하드웨어를 추상화하는 원리
- 장치가 바뀌어도 프로그램 코드가 바뀌지 않는 이유
