# 02 하이퍼바이저와 가상 머신 / 01 VirtualBox 설치와 환경 점검

VirtualBox 7.2.x를 설치하고, 하드웨어 가상화 지원을 확인하며, Extension Pack과 Manager 인터페이스를 점검한다.

## What you will learn

- VirtualBox 7.2.x 설치 (Windows / macOS / Linux)
- 하드웨어 가상화(VT-x/AMD-V) 지원 확인
- Windows Hyper-V 공존과 NEM 모드
- Extension Pack 설치와 라이선스
- VirtualBox Manager 7.2 인터페이스 구성

- `lab01` → VirtualBox 설치, Extension Pack 설치, Manager 인터페이스 탐색
