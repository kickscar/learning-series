# 09 자동화와 운영 심화 / 01 셸 스크립팅

bash 스크립트의 기본 구조, 변수, 조건문, 반복문, 함수를 다루고 운영 자동화 스크립트를 작성한다.

## What you will learn

- 셸 스크립트 기초: shebang, chmod +x, 실행
- 변수: 선언, 참조, 따옴표, 명령 치환 $(), 위치 매개변수, 종료 코드
- 조건문: if/then/else/fi, test/[ ], 숫자·문자열·파일 비교 연산자
- 반복문: for, while
- 함수: 정의, 호출, local, 반환

## Lab

- `lab01/` — 첫 스크립트 (hello.sh)
- `lab03/` — 디스크 사용률 경고 (disk-warn.sh)
- `lab04/` — 서비스 상태 점검 (check-services.sh)
- `lab05/` — 시스템 점검 스크립트 (sys-check.sh)
