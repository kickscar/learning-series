# 06 서비스와 소프트웨어 / 03 패키지 관리

패키지 관리의 구조를 이해하고, dnf와 rpm을 체계적으로 다룬다.

## What you will learn

- 패키지(RPM)와 저장소(repository)의 관계
- rpm(저수준) vs dnf(고수준): 의존성 해결 계층 구조
- dnf: install, remove, update, search, info, provides, repolist
- rpm: -qa, -qi, -ql, -qf로 패키지 조회
- 저장소 구성: /etc/yum.repos.d/, baseos/appstream, EPEL 추가
- Rocky Linux 9에서 yum = dnf 심볼릭 링크
