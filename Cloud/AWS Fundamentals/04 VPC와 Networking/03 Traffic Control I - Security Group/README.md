# 04 VPC와 Networking / 03 Traffic Control I - Security Group

Security Group을 트래픽 제어의 기본 도구로 이해하고, ICMP(ping)와 SSH를 SG 규칙으로 허용/차단해 동작을 체감한다.

## What you will learn

- SG가 ENI 수준의 stateful 방화벽이라는 의미
- Allow 규칙만으로 트래픽을 통제하는 방식
- ICMP/SSH 트래픽을 SG 규칙으로 제어하는 방법

## Examples

- `lab13` -> SG 규칙 변경으로 ICMP/SSH 허용/차단 확인
