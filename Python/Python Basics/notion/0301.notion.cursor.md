# 3.1 내장 스칼라: int, float, str, bytes, bool

## 개요

**스칼라**는 하나의 값으로 다루는 타입이다. Python 내장 스칼라에는 **int**(정수), **float**(실수), **str**(문자열), **bytes**(바이트 열), **bool**(참/거짓)이 있다. 각 타입의 리터럴과 기본 연산, 그리고 문자열 포맷(f-string)과 인코딩(encode/decode) 맛보기를 통해 일상적인 연산과 포맷 출력을 할 수 있게 하는 것이 이 단원의 목표다.

---

## 핵심 개념

### int (정수)

정수는 `42`, `0xff`(16진), `0o10`(8진) 같은 리터럴로 쓴다. Python 3의 int는 **임의 정밀도**라서 메모리가 허용하는 한 자릿수 제한이 없으며, `10 ** 100` 같은 큰 수도 그대로 다룬다. 산술 연산으로는 `+`, `-`, `*`, `//`(몫), `%`(나머지), `**`(거듭제곱) 등을 사용한다. 나눗셈 `/`는 정수끼리 써도 결과가 float이 된다.

```python
a, b = 42, 0xff   # 255
print(a + b)       # 297
print(10 // 3, 10 % 3)   # 몫 3, 나머지 1
print(10 / 3)      # 3.333... (float)
```

### float (실수)

실수는 `3.14`, `1e-2` 같은 리터럴로 쓴다. 내부적으로 IEEE 754 부동소수점을 사용하므로, `0.1 + 0.2`처럼 십진수로 생각한 값과 기계가 표현한 값이 미세하게 다를 수 있다. 따라서 두 실수가 "같다"고 판단할 때는 `==` 대신 `math.isclose(a, b)`를 쓰는 것이 안전하다. 금융이나 정확한 유리수 계산이 필요하면 `decimal.Decimal`, `fractions.Fraction`을 고려한다.

```python
import math
x = 0.1 + 0.2
print(x == 0.3)           # False (부동소수 오차)
print(math.isclose(x, 0.3))  # True
```

### str (문자열)

문자열은 유니코드 코드 포인트의 나열이다. 리터럴은 `'...'`, `"..."`, `'''...'''`(여러 줄 가능)으로 쓴다. `+`로 두 문자열을 이어붙이고, `*`로 같은 문자열을 반복할 수 있다. `in`으로 부분 문자열 포함 여부를 검사하고, 시퀀스이므로 인덱싱 `s[i]`, 슬라이싱 `s[i:j]`도 가능하다. str과 바이트 열(bytes) 사이 변환은 **인코딩**을 통해 이루어진다: `s.encode("utf-8")`은 str을 bytes로, `b.decode("utf-8")`은 bytes를 str로 만든다.

```python
s = "hello"
print(s + "!" * 2)   # hello!!
print("ell" in s)    # True
name, age = "Kim", 20
print(f"{name} is {age}")   # Kim is 20
print(f"{age=}")            # age=20 (디버깅)
print(f"{3.14159:.2f}")      # 3.14
```

### bytes (바이트 열)

bytes는 0~255 범위의 정수(바이트)의 나열이다. 리터럴은 `b'...'` 형태이며, ASCII 범위 문자만 그대로 쓸 수 있다. 한글 등은 str에서 `encode()`로 인코딩한 결과를 bytes로 다룬다. 파일 읽기·쓰기나 네트워크에서는 대부분 bytes 단위로 데이터가 오간다.

```python
text = "가"
raw = text.encode("utf-8")   # bytes
back = raw.decode("utf-8")   # str
print(back == text)          # True
```

### bool (참/거짓)

`True`와 `False` 두 값만 있다. 비교 연산(`<`, `==` 등)이나 논리 연산(`and`, `or`, `not`)의 결과가 bool이 된다. 숫자 문맥에서는 `True`는 1, `False`는 0으로 취급되므로 `True + 1`은 2가 된다. 빈 컨테이너나 0, 빈 문자열 등은 "거짓으로 취급"되어 조건문에서 자주 쓰인다.

```python
print(3 > 1)        # True
print(True + 1)     # 2
print(bool(""), bool([]))   # False False
```

### f-string (포맷 문자열)

문자열 앞에 `f`를 붙이면 중괄호 안에 **표현식**을 넣을 수 있다. `f"{name} is {age}"`처럼 변수 값을 끼워 넣고, `f"{age=}"`처럼 변수 이름과 값을 함께 출력하는 디버깅용 문법도 있다. 소수 자릿수는 `f"{x:.2f}"`처럼 콜론 뒤에 포맷을 지정한다. (위 str 예제에 f-string 포함.)

## 실습 예제

### [실습] lab31

이 실습에서는 **내장 스칼라(int, float, str, bytes, bool)와 f-string**의 개념을 복습한다.

#### ① int — 몫·나머지·제곱, `/` vs `//`

`script01.py`

```python
a, b = 17, 5
print(a / b)
print(a // b, a % b)
print(a ** 2)
```

실행 결과: `3.4` / `3 2` / `289`

#### ② float — `==` vs `math.isclose`

`script02.py`

```python
import math
x, y = 0.1 + 0.2, 0.3
print(x == y)
print(math.isclose(x, y))
```

실행 결과: `False` / `True`

#### ③ str·f-string — 포맷 출력

`script03.py`

```python
name, age, height = "Lee", 25, 1.75
print(f"이름: {name}, 나이: {age}, 키: {height:.2f}m")
```

실행 결과: `이름: Lee, 나이: 25, 키: 1.75m`

#### ④ bytes — encode/decode

`script04.py`

```python
s = "Python"
b = s.encode("utf-8")
print(len(b))
print(b.decode("utf-8") == s)
```

실행 결과: `6` / `True`

#### ⑤ bool — 거짓으로 취급되는 값

`script05.py`

```python
print(bool(0), bool(""), bool([]))
print(bool(1), bool("x"), bool([0]))
```

실행 결과: `False False False` / `True True True`

#### ⑥ 실습 포인트

- **int**: `/`는 float, `//`는 몫, `%`는 나머지. `**`는 거듭제곱.
- **float**: 실수 동등 비교는 `math.isclose` 권장.
- **f-string**: `f"{변수}"`, `f"{변수:.2f}"`로 포맷.
- **bytes**: `str.encode("utf-8")`, `bytes.decode("utf-8")`.
- **bool**: `0`, `""`, `[]` 등은 거짓으로 취급.

## 주의사항 (Pitfalls)

- **float 비교**: `0.1 + 0.2 == 0.3`은 False가 나올 수 있다. 실수 동등 비교는 `math.isclose(a, b)` 사용을 권장.
- **정수 나눗셈**: `/`는 항상 float를 반환. 정수 몫은 `//`, 나머지는 `%`.
- **str vs bytes**: str은 유니코드 문자열, bytes는 바이트 열. 서로 연산하지 않고, `encode`/`decode`로 변환 후 다룬다.
- **빈 값과 bool**: `bool(0)`, `bool("")`, `bool([])`은 False. 0이 아닌 수, 비지 않은 문자열/컨테이너는 True.

---

## Deep Insight

int는 **임의 정밀도**라서 메모리가 허용하는 한 자릿수 제한이 없다. float는 하드웨어 부동소수점이라 표현 범위와 정밀도에 한계가 있고, 그래서 `math.isclose`로 "거의 같음"을 판단하는 것이 안전하다. str은 **유니코드** 코드 포인트의 나열이므로 한 글자당 한 코드 포인트가 아니라 조합 문자 등으로 여러 개일 수 있다. bytes는 str의 한 인코딩(UTF-8 등) 결과로, 파일·네트워크에서는 보통 bytes 단위로 다룬다.

---

## 요약

- 내장 스칼라: **int**(정수), **float**(실수), **str**(문자열), **bytes**(바이트 열), **bool**(참/거짓). 리터럴과 기본 연산을 구분할 수 있어야 한다.
- **float** 비교는 `math.isclose` 권장. **f-string** `f"{x}"`, `f"{x=}"`, `f"{x:.2f}"`로 포맷 출력.
- **str** ↔ **bytes**는 `encode()`/`decode()`로 변환. 정확한 소수 계산이 필요하면 `Decimal`·`Fraction`을 고려한다.
