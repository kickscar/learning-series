# 01 운영체제의 이해 / 02 프로세스와 메모리 모델

프로그램과 프로세스의 차이, 프로세스의 메모리 레이아웃과 생명주기, 멀티태스킹(시분할, 컨텍스트 스위칭), 가상 메모리(페이징, 스왑)를 다룬다.

## What you will learn

- 프로그램(디스크의 파일)과 프로세스(메모리에서 실행 중)의 구분
- 프로세스의 메모리 레이아웃 (Code, Data, Heap, Stack)
- 프로세스 생명주기 (생성 → 준비 → 실행 → 대기 → 종료)
- 시분할과 컨텍스트 스위칭으로 멀티태스킹이 동작하는 원리
- 가상 메모리로 프로세스 간 메모리 격리를 보장하는 원리
