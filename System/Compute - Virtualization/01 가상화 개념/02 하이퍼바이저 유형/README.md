# 01 가상화 개념 / 02 하이퍼바이저 유형

하이퍼바이저의 두 유형(Type-1, Type-2)의 구조와 차이를 이해하고, KVM의 하이브리드 위치와 현재 하이퍼바이저 시장 지형도를 파악한다.

## What you will learn

- Type-1(Bare-Metal)과 Type-2(Hosted) 하이퍼바이저의 구조와 차이
- KVM이 Type-1과 Type-2의 경계에 있는 이유
- Hyper-V의 Root Partition 아키텍처
- 주요 하이퍼바이저의 포지셔닝과 2025-2026 시장 동향
