# 07 Async Network Programming / 02 asyncio

이 디렉토리는 Python에서 **asyncio**와 **async/await**를 이해하기 위한 예제 코드들을 포함한다.

## What you will learn

- asyncio.run() 진입점
- async def, await 기본 사용
- asyncio.gather() 동시 실행
- asyncio.sleep() 대기 시뮬레이션

## Examples

- `async_basic.py` → asyncio.run() 기본 진입
- `async_sleep.py` → await asyncio.sleep() 순차 실행
- `async_gather.py` → asyncio.gather() 동시 실행
