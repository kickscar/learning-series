# Conditional Statements
이 디렉터리의 예제들은 Python의 **조건문과 분기 구조**를 이해하기 위해 구성되어 있다.
`if / elif / else`, 삼항 연산자, 그리고 `and / or`의 short-circuit 동작을 통해 조건에 따른 실행 흐름과 값 선택 방식을 확인한다.

## What These Examples Explain
이 예제들은 다음 개념을 설명한다.

- `if / elif / else` 분기 구조
- Truthy / Falsy 조건 평가
- 삼항 연산자(`a if c else b`)
- `and / or` short-circuit 동작
- `or` 기본값 패턴과 그 한계

## Examples
- `lab01` → 기본 조건 분기
- `lab02` → 삼항 연산자
- `lab03` → `or` 기본값 패턴, `and / or` 평가 순서
