# 01 Cloud와 AWS 시작하기 / 01 Cloud Computing 개요

Cloud Computing의 정의와 가치, 서비스 모델(IaaS/PaaS/SaaS), 배포 모델(Public/Private/Hybrid)을 정리한다. 이후 Chapter에서 AWS 리소스를 연결해 배포 가능한 구조를 만드는 이유를 잡는다.

## What you will learn

- Cloud Computing이 인프라 운영 방식에 주는 변화
- 탄력성(Elasticity), 종량제(Pay-as-you-go), 글로벌 배포(Global Reach)의 의미
- IaaS/PaaS/SaaS 모델과 개발자 관점의 트레이드오프
- Public/Private/Hybrid 배포 모델의 차이
