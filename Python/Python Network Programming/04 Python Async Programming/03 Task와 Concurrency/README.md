# 04 Python Async Programming / 03 Task와 Concurrency

이 디렉토리는 **asyncio.create_task()**, **Task**, **Semaphore**를 이해하기 위한 예제 코드들을 포함한다.

## What you will learn

- asyncio.create_task()로 Task 직접 생성
- gather() vs create_task() 차이
- asyncio.Semaphore로 동시 실행 수 제한

## Examples

- `lab01.py` → create_task + gather
- `lab02.py` → Semaphore로 동시 2개 제한
