# 03 셸과 CLI / 03 파일 내용 다루기

파일의 내용을 보고, 텍스트를 검색하고, 데이터를 가공하는 도구를 다룬다. 서버 운영에서 설정 파일 확인과 로그 분석의 기본이다.

## What you will learn

- cat, less(페이저), head, tail로 파일 내용 보기
- tail -f로 로그 실시간 추적
- grep으로 패턴 검색 (-i, -r, -n, -v)과 정규표현식 기초 (., *, ^, $, [])
- sort, uniq, wc, cut, tr로 텍스트 가공
- 와일드카드(셸, 파일 이름)와 정규표현식(grep, 파일 내용)의 차이
