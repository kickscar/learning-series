# Chapter Definition

## 왜 필요한가

Terraform configuration을 재사용 가능하게 만들려면 variable, tfvars, output, local value를 체계적으로 사용해야 한다. 이 Chapter는 코드 재사용성과 환경별 차이를 다루는 기초를 만든다.

# Modern Key Concepts

- **Input Variable**
  입력 인터페이스
- **Output**
  결과값 노출 메커니즘
- **Local Value**
  내부 계산과 중복 제거 수단
- **tfvars**
  환경별 설정 파일
