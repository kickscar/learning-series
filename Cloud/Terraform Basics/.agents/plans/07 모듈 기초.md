# Modern Key Concepts

- **Module**
  재사용 가능한 infrastructure 구성 단위
- **Root Module**
  실행 진입점
- **Child Module**
  재사용되는 하위 구성
- **Module Versioning**
  안정적 재사용을 위한 버전 관리
