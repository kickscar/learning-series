# 01 Cloud와 GCP 시작하기 / 01 Cloud Computing 개요

Cloud Computing의 등장 배경과 On-premise와의 차이를 이해하고, IaaS/PaaS/SaaS 서비스 모델과 GCP의 핵심 서비스 영역을 파악한다.

## What you will learn

- On-premise의 한계(초기 비용, 과잉 프로비저닝, 운영 부담)와 Cloud Computing이 해결하는 문제
- Cloud의 핵심 특성: 탄력성(Elasticity), 종량제(Pay-as-you-go), 글로벌 배포
- IaaS / PaaS / SaaS 서비스 모델의 차이와 개발팀 관리 범위
- GCP가 제공하는 핵심 서비스 영역 (Compute, Networking, Storage, Database)
