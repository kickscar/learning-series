## Assignment Statements
이 디렉터리의 예제들은 **Python 대입문이 실제로 무엇을 하는지**를 “이름과 값의 연결”이라는 관점에서 이해하기 위한 코드들이다.
`a = 1`, `a = b`, `a = a + 1`이 각각 어떤 의미를 가지는지와 대입이 표현식이 아니라 문(statement)이라는 점에 초점을 둔다.

## What These Examples Explain
이 예제들은 다음 개념을 설명하기 위해 구성되어 있다.

- 대입문의 기본 형식: `이름 = 표현식`
- 오른쪽을 먼저 계산하는 실행 순서
- 자기 자신을 사용하는 대입의 의미
- 이름 간 대입(`a = b`)의 실제 동작
- 연쇄 대입(`a = b = 0`) 패턴

## Examples
- `lab01` → 가장 단순한 대입, `a = a + 1` 실행 흐름
- `lab02` → `a = b`의 의미
- `lab03` → 연쇄 대입 패턴
