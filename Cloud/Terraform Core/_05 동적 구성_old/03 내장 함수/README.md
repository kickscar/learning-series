# Ch05 동적 구성 / 03 내장 함수

내장 함수를 카테고리별로 정리하고, Sec01~02의 모든 리소스를 통합해 Custom VPC를 완성한다. 설정 검증을 내장 함수로 강화한다.

## What you will learn

- `cidrsubnet`으로 서브넷 CIDR 자동 계산 — 하드코딩 제거
- `lookup`, `try`, `can`으로 설정 누락/오류 방어
- `startswith`, `contains`, `keys`로 서브넷 유형 판별 + dynamic route 설정
- for expression으로 동적 output — 설정 변경에 자동 대응
- String, Collection, Numeric, Encoding, Type Conversion 함수 카테고리 정리

## Lab

- **lab01**: Custom VPC 완성 — Sec01~02 전체 통합 + 내장 함수로 검증 강화 (Ch05 결정판, Ch06 모듈화 기반)
