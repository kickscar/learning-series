# 02 설치와 초기 설정 / 02 Rocky Linux 설치

VirtualBox VM을 생성하고 Rocky Linux Minimal ISO로 부팅하여 수동 파티셔닝으로 설치를 완료한다.

## What you will learn

- VirtualBox VM 생성 (UEFI, CPU 2코어, 메모리 2 GB, 디스크 40 GB)
- Anaconda 설치 프로그램의 Hub-and-Spoke 구조
- Custom → Standard Partition으로 수동 파티셔닝
- root 비밀번호 설정과 일반 사용자(wheel 그룹) 생성
- 설치 후 lsblk, df -hT, free -h로 결과 확인
