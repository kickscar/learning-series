# Basic Operators
이 디렉터리의 예제들은 Python의 **기본 연산자(operator)** 들의 의미와 동작 방식을 이해하기 위해 구성되어 있다.
산술, 비교, 논리, 할당 연산자와 함께 연산자 우선순위가 실제 실행 결과에 어떤 영향을 주는지를 간단한 예제로 확인한다.

## What These Examples Explain
이 예제들은 다음 개념을 설명한다.

- 산술 연산자(`/`, `//`, `%`, `**`)의 차이
- 비교 연산자의 bool 반환값
- 논리 연산자의 short-circuit 동작
- 할당 연산자(`+=`, `*=`)의 의미
- 연산자 우선순위와 괄호 사용

## Examples
- `lab01` → 산술 연산자
- `lab02` → 비교 연산자
- `lab03` → 논리 연산자와 반환값
- `lab04` → 할당 연산자
- `lab05` → 연산자 우선순위
