# 08 ECS - Container Orchestration / 01 ECR과 Container Image

ECR(Elastic Container Registry)을 Container Image 저장소로 사용하고, ECS 배포를 위한 "Image Push" 흐름을 구성한다.

## What you will learn

- ECR이 컨테이너 배포 흐름에서 갖는 위치(Build, Ship, Run)
- ECR Repository URI, Tag, Digest의 의미와 차이
- Image Scan, Lifecycle Policy 같은 운영 포인트

## Examples

- `lab22` -> ECR Repository 생성, Gallery Image 빌드/Tag/Push, Console에서 Image 확인
