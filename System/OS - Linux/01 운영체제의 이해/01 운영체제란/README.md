# 01 운영체제의 이해 / 01 운영체제란

운영체제가 없으면 어떤 일이 벌어지는지에서 출발하여, OS가 하드웨어를 어떻게 추상화하고 관리하는지, 커널과 유저 공간이 어떻게 나뉘는지를 다룬다.

## What you will learn

- OS가 없을 때의 세 가지 문제 (자원 충돌, 하드웨어 종속, 동시 실행 불가)
- OS의 두 가지 역할: 자원 추상화와 자원 관리
- 커널 모드와 유저 모드의 분리, 시스템 콜의 역할
- 하드웨어 → 커널 → 시스템 콜 → 유저 공간의 전체 계층 구조
