# Cloud Workloads

`Cloud/` 트랙에서 **실제로 실행해 볼 수 있는 샘플 애플리케이션(Workload)** 을 모아 둔 디렉터리다. 시리즈별 문서·소규모 예제와는 구분되어, **배포·실습용 앱 코드**가 여기에 올라온다.

---

## 개요

- **역할**: 여러 학습 시리즈에 걸쳐 공통으로 쓰이거나, 특정 Lab·시나리오의 **실행 대상**이 되는 애플리케이션을 한곳에서 관리한다.
- **범위**: 사용 언어·런타임·클라우드는 고정하지 않는다. 현재는 Java / Spring Boot 기반 예제가 포함되어 있다.

---

## Cloud Series와의 관계

`Cloud/` 아래에는 주제별 **학습 시리즈**가 디렉터리 단위로 있다. 이 디렉터리의 앱들은 필요에 따라 해당 시리즈의 실습·배포 시나리오와 맞춰 사용할 수 있다.

| 시리즈 | 관계 (요약) |
|--------|----------------|
| **AWS Fundamentals** | **gallery-spring-boot**는 시리즈에서 다루는 Gallery 예제 애플리케이션에 해당한다. |
| **AWS IaC with CloudFormation** | 인프라를 코드로 올린 뒤 같은 앱을 배포하는 흐름과 조합할 수 있다. |
| **Terraform Core** | 필요 시 Provider 실습·배포 예제와 함께 참조할 수 있다. |
| **기타 (GCP 등)** | 시리즈가 확장되면 추가 샘플 앱이 이 디렉터리에 포함될 수 있다. |
