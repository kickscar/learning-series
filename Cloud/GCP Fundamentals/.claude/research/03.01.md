## 버전 및 서비스 정보

| 항목 | 현재 기준 | 비고 |
|------|----------|------|
| Machine Type 시리즈 | E2, N2, N2D, N4, C2, C3, M2, M3 등 | E2가 비용 효율 범용, 실습에 주로 사용 |
| 실습 권장 Machine Type | e2-medium (2 vCPU, 4GB) | 무료 계층: f1-micro (e2-micro) |
| Preemptible VM | Spot VM으로 통합 중 | Preemptible은 레거시, Spot VM 권장 |

## 조사 메모

- 순수 이론 섹션, 콘솔 경로 불필요
- AWS EC2와의 비교가 주요 내용 — 독자가 AWS 경험 있을 수 있음
- Machine Type 명명 규칙: {시리즈}-{vCPU 수} (예: e2-medium = E2 시리즈, 중간 사양)
- Preemptible VM은 최대 24시간 실행, GCP가 언제든 회수 가능 → Spot VM으로 점진 통합
