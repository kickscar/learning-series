| 항목 | 값 |
|------|---|
| Runtime | Java 21 |
| Framework | Spring Boot 3.5.x |
| View | Thymeleaf |
| DB | H2 인메모리 (기본) |
| Storage | 로컬 디스크 (기본) |

## Ch02 Gallery 설정

| Terraform 변수 | 앱 설정 | 값 |
|---------------|---------|---|
| `var.web_port` | `server.port` | `8080` |
| `var.instance_type` | — | `t3.small` (빌드 메모리 여유) |
| `var.env` | `spring.profiles.active` | `dev` |

프로파일 `dev`: 포트 8080, H2 인메모리, 로컬 스토리지, Thymeleaf 캐시 끔.

## EC2 수동 설치 절차

Ch02 Gallery는 user_data 자동화 없이 **SSM으로 접속해 수동 설치**한다.

### 1. JDK 21 + git 설치

```bash
sudo dnf install -y java-21-amazon-corretto-headless git
```

### 2. 소스 클론 (sparse checkout)

```bash
git clone --filter=blob:none --sparse https://github.com/kickscar/learning-series.git ~/workspace
cd ~/workspace
git sparse-checkout init --no-cone
git sparse-checkout set Cloud/Workloads/gallery-spring-boot
cd Cloud/Workloads/gallery-spring-boot
```

### 3. Maven 빌드

```bash
chmod +x ./mvnw
./mvnw clean package -DskipTests -Dbuild.finalName=gallery
```

산출물: `target/gallery.jar`. t3.small 이상 필요 (Maven 빌드 메모리).

### 4. 실행

```bash
java -jar target/gallery.jar
```

foreground 실행. Ctrl+C로 종료.

### 5. 접속 확인

`http://{public_ip}:8080`

## user_data 자동화 시점

Ch02 Gallery는 수동 설치. user_data + templatefile + systemd 자동화는 **Ch04 Gallery**부터 도입.
이유: 설치 절차를 직접 경험한 후 자동화해야 의미가 있다.
