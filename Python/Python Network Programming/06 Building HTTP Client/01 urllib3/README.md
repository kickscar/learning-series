# 05 Building HTTP Client / 01 urllib3

이 디렉토리는 Python에서 **urllib3**를 이해하기 위한 예제 코드들을 포함한다.

## What you will learn

- PoolManager를 통한 연결 풀 관리 및 HTTP 요청
- HTTPResponse의 status, data, headers, json()으로 응답 처리
- GET, POST(form), POST(JSON) 요청 방법

## Examples

- `lab01.py` → PoolManager로 GET 요청 수행
- `lab02.py` → POST 요청 (form 데이터)
- `lab03.py` → POST 요청 (JSON 본문)
