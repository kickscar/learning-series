# 06 Building HTTP Server / 01 http.server

이 디렉토리는 Python에서 **http.server**를 이해하기 위한 예제 코드들을 포함한다.

## What you will learn

- SimpleHTTPRequestHandler로 정적 파일 서빙
- HTTPServer(TCPServer)와 Handler 조합
- 특정 디렉터리 서빙 (directory 인수)

## Examples

- `simple_server.py` → 기본 Simple HTTP Server 구동
- `simple_server_directory.py` → 지정 디렉터리 서빙
