## 3. 세부 주제 상세

### Ch2: 변수 (basic 입문자)

- **목표**: “변수”를 basic한 입문자 수준에서 정의하고, 이름·대입·사용만 다룸. 객체/참조/`id`/`is`는 Ch5에서.
- **세부 주제 (제안)**  
  1. 변수란 — 이름과 값의 대응  
  2. 대입문 (`a = 1`, `a = b`)  
  3. 이름 사용 — 읽기/쓰기  
  4. 변수와 타입 — 값에 따른 타입 (구체적 타입은 Ch3에서)  
  5. (선택) 여러 개 대입 — `a, b = 1, 2` 등  
- **비고**: 기존 plan에 “변수 전용 챕터”가 없으므로, 위는 **신규 설계 초안**. 컨펌 시 확정.

  → 기존 §2.4 활용

## 7. 문서 작성 규칙 (notion.cursor.md)

- **파일명**: `{챕터2자리}{주제2자리}.notion.cursor.md` (예: 0301, 0402).
- **구조**: 개요 → **핵심 개념**(개념 설명용 코드가 필요하면 해당 소제목 안에 추가) → **실습 예제 (lab{n})** → 주의사항 (Pitfalls) → Deep Insight (→ 요약은 선택).
- **핵심 개념 (Ch3부터 적용)**: 요약형 불릿이 아니라 가능하면 자세히 서술한다. 핵심 개념 **설명을 위한 코드**는 해당 소제목 안에 넣어 충실히 제시한다. 4장부터 10장까지 동일 방식 적용.
- **실습 예제**: **lab{n}** 넘버링. 핵심 개념과 **1:1 대응**하는 개념 이해 **복습 코드** 수준. 형식: `# [실습] lab{n}` → 소개문 → ① ② ③ (스크립트명 + 코드 + 실행 결과) → **실습 포인트** (불릿 정리). 자세한 형식은 `.cursorrules` §3 참고.
