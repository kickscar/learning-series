# Modern Key Concepts

Terraform을 현대적인 Infrastructure 도구로 이해하기 위해 다음 핵심 개념을 강조한다.

- **Terraform Workflow**
  init → plan → apply → destroy

- **Terraform State**
  infrastructure 상태 관리 메커니즘

- **Dependency Graph**
  resource 간 자동 의존성 관리

- **Module Architecture**
  재사용 가능한 infrastructure 설계

- **Provider Ecosystem**
  multi-cloud infrastructure 지원

- **Terraform vs CloudFormation**
  IaC 접근 방식 비교

- **HCL (HashiCorp Configuration Language)**
  Terraform configuration language
