# 01 시작하기 / 04 실행 진입점

`if __name__ == "__main__"` 패턴의 실무 사용법과 `__main__.py`로 패키지를 실행 가능하게 만드는 방법을 다룬다.

## What you will learn

- `if __name__ == "__main__"` 패턴으로 모듈을 라이브러리와 스크립트 양쪽으로 사용하는 방법
- `main()` 함수 컨벤션과 `sys.exit()` 종료 코드 규칙
- `__main__.py`의 역할 — 패키지에 실행 진입점 부여
- `__init__.py`와 `__main__.py`의 실행 순서
- `-m` 옵션과 상대 import의 관계

## Examples

- `lab01-entry-point/` → 단일 모듈의 실행/import 구분, 패키지 `__main__.py`로 `-m` 실행
