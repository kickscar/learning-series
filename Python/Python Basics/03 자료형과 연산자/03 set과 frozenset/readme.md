## Set and Frozenset
이 디렉터리의 예제들은 Python의 **집합 타입(set, frozenset)** 의 기본 개념과 사용 패턴을 이해하기 위해 구성되어 있다.
중복 제거, 빠른 멤버십 검사, 그리고 집합 연산의 특성을 코드를 통해 직접 확인하는 것이 목적이다.

## What These Examples Explain
이 예제들은 다음 개념을 설명한다.

- set과 frozenset의 차이 (가변 vs 불변)
- 중복 제거와 멤버십 검사 활용
- 집합 연산 (`|`, `&`, `-`, `^`)
- hashable 요소 제약
- 빈 set 생성 규칙 (`set()`)

## Examples
- `lab01` → 중복 제거와 멤버십 검사
- `lab02` → 빈 set 생성
- `lab03` → `add`, `remove`, `discard`
- `lab04` → 불변 집합과 키 사용
- `lab05` → 합집합, 교집합, 차집합, 대칭차
- `lab06` → 단일 요소 set 생성
