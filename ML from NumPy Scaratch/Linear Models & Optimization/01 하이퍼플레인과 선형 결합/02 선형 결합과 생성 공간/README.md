# 02 선형 결합과 생성 공간

벡터의 선형 결합, Span, 선형 독립, 기저, 랭크를 다루고, $Xw$가 $X$의 열 벡터들의 선형 결합이라는 핵심 연결을 짚는다.

## 학습 내용

- 선형 결합: $a_1 v_1 + a_2 v_2 + \cdots + a_k v_k$
- Span: 선형 결합으로 도달할 수 있는 모든 점의 집합
- 선형 독립과 종속, 기저(Basis)의 직관
- $\text{rank}(X) = d$이면 정규방정식의 유일한 해가 존재 (Ch02 복선)
