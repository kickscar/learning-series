# 03 Compute - Compute Engine / 01 Compute Engine 개요

Compute Engine VM이 어떤 구성 요소로 이루어지는지, 사양은 어떻게 정의하는지, 디스크와 이미지는 어떤 개념인지 이해한다.

## What you will learn

- Compute Engine VM 구성 요소 (VM Instance, NIC, Persistent Disk, External IP, Firewall Rule)와 연결 구조
- Machine Type 체계와 명명 규칙 — E2/N2/C2 시리즈, e2-medium 표기 방식
- VM Image 유형 (Public Image, Custom Image, Marketplace)과 선택 기준
- Persistent Disk 유형 (Standard HDD, Balanced PD, SSD PD)과 Boot Disk / Data Disk 분리 전략
- AWS EC2와의 비교 — Firewall Rule(Network Tag 기반) vs Security Group의 핵심 차이
