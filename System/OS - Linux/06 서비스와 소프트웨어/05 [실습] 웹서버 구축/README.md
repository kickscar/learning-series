# 06 서비스와 소프트웨어 / 05 [실습] 웹서버 구축

Ch02~Ch06에서 배운 기술을 종합하여 Nginx 웹서버를 구축한다.

## What you will learn

- dnf로 Nginx 설치 (06.03)
- systemctl로 서비스 시작·부팅 연동 (06.01, 06.02)
- HTML 파일 배포와 curl로 동작 확인
- nginx -t로 설정 검증 후 reload
- journalctl과 /var/log/nginx/로 로그 확인 (06.04)
- ps로 master/worker 프로세스 구조 확인 (05.01)
