# status/report — 진행 현황 보고

## 목적

시리즈 전체 진행 현황을 파악하고 보고한다.

## 실행 단계

### Step 1: series-hierarchy.md 읽기

- 챕터/섹션 구조와 상태 컬럼 확인

### Step 2: .claude/draft/ 스캔

- 실제 draft 파일 존재 여부 확인
- 파일명 패턴: `{ch_no}.{sec_no}.claude.notion.md`

### Step 3: progress.md 읽기 (있으면)

- 상세 진행 기록 확인

### Step 4: 현황 보고 출력

```
## 진행 현황 보고

| Chapter | 섹션 수 | 완료 | 진행중 | 미시작 | 상태 |
|---------|--------|------|--------|--------|------|
| 01 | 4 | 0 | 0 | 4 | 계획수립 |
| 02 | 4 | 0 | 0 | 4 | 계획수립 |
| ... | ... | ... | ... | ... | ... |

### 요약
- 전체: {n}개 섹션 ({n}개 Gallery 포함)
- 완료: {n}개
- 진행중: {n}개
- 미시작: {n}개
```

## 규칙

- 채팅 전용 출력 (파일 수정 없음)
