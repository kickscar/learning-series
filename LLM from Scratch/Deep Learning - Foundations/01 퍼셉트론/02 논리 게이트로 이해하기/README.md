# 02 논리 게이트로 이해하기

AND · OR · NAND — 같은 퍼셉트론 구조에서 가중치와 편향만 바꾸면 서로 다른 논리 게이트가 된다.

## 학습 내용

- AND · OR · NAND 게이트의 진리표와 퍼셉트론 파라미터 매핑
- 결정 경계 시각화로 파라미터 변화의 기하학적 의미 확인
- NAND = AND의 파라미터 부호 반전
- "같은 구조, 다른 파라미터" 원칙
