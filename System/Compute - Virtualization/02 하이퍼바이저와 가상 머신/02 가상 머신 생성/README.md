# 02 하이퍼바이저와 가상 머신 / 02 가상 머신 생성

VM의 구성 요소와 자원 할당 원리를 이해하고, VirtualBox에서 첫 VM을 생성하여 Rocky Linux를 설치한다.

## What you will learn

- 가상 머신의 구성 요소와 설정 파일 구조
- CPU Time-sharing, 메모리 할당, 네트워크 기본 모드(NAT)의 원리
- VM 생성 마법사의 4단계 과정
- Rocky Linux 설치와 Guest Additions 설치
- VBoxManage CLI로 VM 구성을 확인하는 방법

## Examples

- `lab01` → 첫 VM 생성, Rocky Linux 설치, Guest Additions 설치
