# 05 Building HTTP Client / 03 httpx

이 디렉토리는 Python에서 **httpx**의 동기·비동기 HTTP 클라이언트를 이해하기 위한 예제 코드들을 포함한다.

## What you will learn

- httpx의 동기 API (get, post, Client)
- AsyncClient를 통한 비동기 요청
- asyncio.gather로 여러 요청 동시 수행

## Examples

- `lab01.py` → 동기 GET 요청
- `lab02.py` → 동기 POST 요청 (JSON)
- `lab03.py` → Client 사용 (연결 풀)
- `lab04.py` → AsyncClient로 비동기 요청
- `lab05.py` → asyncio.gather로 여러 요청 동시 수행
