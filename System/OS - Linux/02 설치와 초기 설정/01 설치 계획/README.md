# 02 설치와 초기 설정 / 01 설치 계획

Rocky Linux를 설치하기 전에 결정해야 할 것들을 다룬다. 파티션 설계, 파일시스템 선택, 네트워크 구성 계획, 계정 설정 계획을 세운다.

## What you will learn

- Rocky Linux의 RHEL 호환성과 버전 체계 (9.x, EOL 정책)
- UEFI 부팅과 파티션 구성 (/boot/efi, /boot, swap, /home, /)
- Standard Partition vs LVM의 차이와 이 시리즈에서 Standard Partition을 선택하는 이유
- xfs(Rocky 기본)와 ext4의 비교
- NetworkManager 전면 전환, ifcfg deprecated 등 RHEL 9 주요 변경사항
