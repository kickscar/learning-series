# 04 선형 모델의 일반형

디자인 행렬 $X$의 구조를 이해하고, 1-column 추가 트릭으로 절편을 흡수하며, 순전파 $\hat{y} = Xw$를 NumPy로 구현한다.

## 학습 내용

- 디자인 행렬(Design Matrix) $X$: $(n, d)$ 행렬, 각 행이 하나의 데이터
- 절편 처리: 1-column 추가로 $b$를 $w$에 흡수
- 순전파 구현과 shape 추적
- 2D 데이터에서 예측 직선 시각화
