# 03 하이퍼플레인의 정의와 기하학적 해석

$w^T x + b = 0$이 공간에서 만드는 도형을 이해하고, 법선 벡터와 점-하이퍼플레인 거리 공식을 유도·구현·시각화한다.

## 학습 내용

- 하이퍼플레인: $d$차원 공간을 둘로 나누는 $(d-1)$차원 경계면
- $w$가 법선 벡터임을 증명
- 점에서 하이퍼플레인까지의 부호 있는 거리: $\frac{w^T x_0 + b}{\|w\|}$
- 2D(직선)와 3D(평면)에서의 시각화
