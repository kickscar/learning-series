## 사용 규칙

- 챕터명/섹션명 헤딩 없이 시작 (Notion 타이틀이 대신)
- 첫 H1 전에 **개요 단락** (1~3줄): 앞 섹션 환기 + 이번 섹션 예고
- `# {개념명}` — H1은 번호 없음, 볼드 없음
- 헤딩에 볼드(`**`) 처리 안 함
- 기술 용어: 영문 기준, 첫 등장 시 한글 병기 `State (상태 파일)`
- HCL 코드: ` ```hcl ` 언어 지정 필수
- 버전 요구 기능: 본문에 `(TF 1.5+)` 명시
