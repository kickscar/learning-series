# 01 스칼라에서 벡터로

변수 하나의 회귀에서 출발해, 변수가 늘어나면 왜 선형대수가 필요한지를 체감하고, 시리즈 전체에서 사용할 표기법을 도입한다.

## 학습 내용

- 단일 변수 회귀 $y = wx + b$에서 다변수 $\hat{y} = Xw + b$로의 확장
- 벡터와 행렬이 "규모가 커져도 형태가 바뀌지 않는 언어"인 이유
- 시리즈 전체 표기법: $X$, $w$, $b$, $y$, $\eta$, $\lambda$, $\nabla$
