# 06 Building HTTP Client / 04 aiohttp

이 디렉토리는 Python에서 **aiohttp** 비동기 HTTP 클라이언트를 이해하기 위한 예제 코드들을 포함한다.

## What you will learn

- aiohttp ClientSession으로 비동기 HTTP 요청
- GET, POST(JSON) 요청
- asyncio.gather()와 결합한 동시 요청

## Examples

- `client_get.py` → GET 요청
- `client_post.py` → POST 요청 (JSON)
- `client_concurrent.py` → asyncio.gather() 동시 요청
