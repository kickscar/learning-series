# Reading vs Writing Names
이 디렉터리의 예제들은 **Python에서 이름(name)이 읽히는 경우와 쓰이는(대입되는) 경우의 차이**를 이해하기 위한 코드들이다.
같은 이름이라도 표현식에서 사용될 때와 대입문에서 사용될 때의 역할이 어떻게 달라지는지에 초점을 둔다.


## What These Examples Explain
이 예제들은 다음 개념을 설명하기 위해 구성되어 있다.

- 이름을 읽을 때 값이 사용되는 방식
- 대입문이 이름에 새 값을 붙이는 과정
- 같은 이름에 재대입할 때 일어나는 변화
- 정의되지 않은 이름을 읽을 때 발생하는 `NameError`
- 읽기/쓰기 구분이 스코프 이해로 이어지는 이유

## Examples
- `lab01` → 표현식에서 이름 읽기
- `lab02` → 같은 이름에 다시 대입하기
- `lab03` → 정의 전 이름 사용 시 오류
