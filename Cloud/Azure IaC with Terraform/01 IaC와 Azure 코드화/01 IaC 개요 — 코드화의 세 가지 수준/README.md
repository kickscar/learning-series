# 01 IaC와 Azure 코드화 / 01 IaC 개요 — 코드화의 세 가지 수준

IaC의 본질과 코드화 스펙트럼(CLI → SDK → 프로비저닝 도구)을 이해하고, 이 시리즈가 어디에 집중하는지 정리한다.

## What you will learn

- IaC는 Terraform만이 아니다 — CLI 스크립트도, SDK도 코드화의 일부다
- 코드화의 세 가지 수준: CLI / 스크립트, SDK, 프로비저닝 도구
- 각 수준의 강점과 한계, 실무에서의 조합 패턴
- 이 시리즈의 범위: Terraform 주력, az CLI 결과 확인, SDK 개념 이해
