# Iteration and the for Statement
이 디렉터리의 예제들은 Python의 `for` 문과 이터러블(iterable) / 이터레이터(iterator) 개념을 이해하기 위해 구성되어 있다.
`for` 문이 내부적으로 `iter()`와 `next()`를 호출하여 반복을 수행한다는 점을 코드로 확인한다.

## What These Examples Explain
이 예제들은 다음 개념을 설명한다.

- `for` 문의 내부 동작
- iterable vs iterator 차이
- `iter()` 와 `next()`의 수동 사용
- `range`, `dict`, `enumerate` 순회 방식
- 이터레이터 소진과 재사용 불가 특성

## Examples
- `lab01` → list와 str 순회
- `lab02` → `iter()` 와 `next()`
- `lab03` → `range`, `enumerate`
- `lab04` → dict 순회 (keys / items)
