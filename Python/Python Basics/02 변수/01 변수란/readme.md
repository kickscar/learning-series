# Variables: Names Bound to Values
이 디렉터리의 예제들은 **Python에서 변수란 무엇인가**를 “이름과 값의 대응 관계”라는 관점에서 이해하기 위한 코드들이다.

변수를 값이 들어 있는 상자가 아니라, 값을 가리키는 이름으로 이해하는 데 초점을 둔다.

## What These Examples Explain
이 예제들은 다음 개념을 설명하기 위해 구성되어 있다.

- 변수는 값이 아니라 **이름**이라는 개념
- 대입(assign)이 곧 이름의 정의라는 점
- 대입 전·후 이름 사용의 차이
- 식별자(identifier) 규칙
- 오타와 정의되지 않은 이름이 `NameError`를 만드는 이유

## Examples
- `lab01` → 올바른 식별자 예시
- `lab02` → 잘못된 식별자 예시
- `lab03` → 대입 전·후 이름 사용 비교
