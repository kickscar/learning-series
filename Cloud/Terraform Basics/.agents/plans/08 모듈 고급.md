# Modern Key Concepts

- **Module Composition**
  여러 module을 조합한 구조
- **Infrastructure Layering**
  계층별 책임 분리
- **Root Module Separation**
  환경별 진입점 분리
- **Reusable Interface**
  재사용 가능한 입력/출력 계약
