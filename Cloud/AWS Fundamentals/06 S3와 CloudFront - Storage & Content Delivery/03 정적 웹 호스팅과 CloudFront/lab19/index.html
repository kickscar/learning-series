Hello S3 Web

Amazon S3 정적 웹 호스팅과 CloudFront로 서비스할 때 쓸 수 있는 예제 페이지입니다.

이 폴더 전체(index.html, error.html, css/, images/)를 Bucket에 업로드합니다.

S3 Website hosting에서 Error document에 error.html을 지정하면, 오류에 해당하는 요청에 대해 리다이렉트가 아니라 그 HTML이 응답 본문으로 대체되어 보일 수 있습니다. error.html은 외부 CSS·이미지 없이 단독으로 두는 예제입니다.

  • 상대 경로만 사용했으므로 디렉터리 구조를 유지한 채 업로드하세요.
  • 배포 후 Website endpoint 또는 CloudFront 도메인으로 접속해 확인합니다.
