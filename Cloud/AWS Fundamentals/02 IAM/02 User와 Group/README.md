# 02 IAM / 02 User와 Group

IAM User와 Group의 역할을 권한 관리 체계 관점에서 정리한다. `lab03`에서 Admin Group을 만들고 IAM User를 생성해 Group으로 권한을 부여한 뒤, MFA를 설정하고 IAM User로 Console 로그인까지 확인한다.

## What you will learn

- IAM User의 Console 접근과 Programmatic 접근(Access Key)의 차이
- User에 직접 권한을 붙이는 방식의 한계와 Group 기반 관리의 이유
- Group-Policy-User 연결 구조와 운영 관점
- MFA의 필요성과 IAM User에 MFA를 적용하는 흐름

## Examples

- `lab03` → Admin Group 생성, IAM User 생성/Group 추가, MFA 설정, IAM User Console 로그인
