# Values and Types
이 디렉터리의 예제들은 **Python에서 타입이 변수에 있는 것이 아니라 값에 있다는 개념**을 이해하기 위한 코드들이다.
같은 이름에 서로 다른 타입의 값을 대입할 수 있다는 점과, `type(x)`가 무엇을 의미하는지를 실습을 통해 확인한다.

## What These Examples Explain
이 예제들은 다음 개념을 설명하기 위해 구성되어 있다.

- 변수 자체에는 타입이 고정되어 있지 않다는 점
- 같은 이름에 서로 다른 타입의 값을 붙일 수 있는 이유
- `type(x)`가 “변수의 타입”이 아니라 “값의 타입”을 반환한다는 점
- 값의 타입에 따라 허용되는 연산이 달라진다는 사실

## Examples

- `lab01` → 같은 이름에 다른 타입의 값 대입
- `lab02` → `type(x)`가 현재 값 기준임을 확인
