# Ch01 Terraform 시작하기 / 02 설치 및 환경 구성

Terraform CLI를 설치하고 AWS Provider와 연결하는 환경을 구성한다. 이후 모든 실습의 기반이 된다.

## What you will learn

- tfenv로 Terraform 버전을 관리하고 1.14.x를 설치하는 방법
- `required_providers`와 `provider` 블록으로 AWS Provider를 선언하고 연결하는 구조
- AWS credentials 설정 방법 (환경변수 / `~/.aws/credentials`)

## Examples

- `lab01` → terraform CLI 설치 및 버전 확인 (1.10.x), AWS credentials 설정 확인, provider 블록 작성 후 `terraform init` 실행, `.terraform/` 디렉토리 구조 확인
