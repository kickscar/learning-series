# Chapter 01 — Cloud와 Azure 시작하기

## 목표

Cloud Computing의 기본 개념과 Azure의 글로벌 인프라를 이해하고,
Azure 계정을 생성해 Portal과 Cloud Shell을 직접 다뤄본다.

  - Cloud의 핵심 특성 (탄력성, 종량제, 글로벌 배포)

- **유형**: 이론 + 실습
