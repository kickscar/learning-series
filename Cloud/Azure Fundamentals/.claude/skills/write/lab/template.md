## 사용 규칙

- 이론 파트 뒤에 `# [실습] lab{no}: {제목}` H1으로 실습 시작 — 이론/실습 경계 구분 + Notion 페이지 분리 기준
- ⚠️ 비용 주의 섹션 없음
- `# 1. 전체 아키텍처`, `# 2. 사전 준비` 고정
- `# 3.` 이후 각 작업을 직접 H1으로 명명 — 고정 이름 없음, 작업 내용에 맞게
- 콘솔 스텝: `## {n}. Azure Portal > {서비스} > **{최종 화면}**` 형태 권장
- 이미지: `[콘솔화면: Azure Portal > 경로 > 화면 설명]`
- 설정값: **볼드 + 백틱** → **`{값}`**
- 리소스명 규칙: `{type}-{context}` (예: `rg-lab03`, `vnet-gallery`)
