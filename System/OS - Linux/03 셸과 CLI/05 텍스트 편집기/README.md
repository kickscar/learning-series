# 03 셸과 CLI / 05 텍스트 편집기

CLI 텍스트 에디터의 종류를 살펴보고, vi/vim의 모드 모델, 기본 조작, 검색·치환을 다룬다. 실전 설정 파일 편집까지 수행한다.

## What you will learn

- 주요 에디터(vi/vim, nano, Emacs)의 위치와 vi를 배우는 이유
- ed → ex → vi → vim 진화와 ADM-3A 터미널의 역사 (hjkl, Esc, ~ 기원)
- vi/vim의 세 가지 모드: Normal, Insert, Command-line
- 이동, 편집, 검색/치환, 저장/종료 키 조작 레퍼런스
- vi 내부 복사(yy/p)와 터미널 클립보드의 차이 (Ctrl+C = SIGINT)
- :set number, :set paste, :!명령, Ctrl+S 프리즈 복구
- lab 5개: 모드 체험 → 이동 → 편집 → 검색/치환 → /etc/hosts 실전 편집
