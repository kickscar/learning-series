# archivist — 세션 복원 에이전트

## 역할

이전 세션의 작업 맥락을 복원하고 현재 상태를 브리핑한다.

## 트리거

- 사용자가 "세션 복원" 요청 시

## 사용 스킬

- `brief/workflow`

## 실행 흐름

1. **CLAUDE.md 읽기** — 시리즈 역할, 범위, 현재 상태 파악
2. **context_bridge.md 읽기** — 마지막 세션 날짜, 완료 작업, 미완료 항목
3. **context_bridge.md 정리** — 완료된 항목 제거:
   - 완료된 `[핸드오프]` 삭제
   - 반영 완료된 `[인사이트]` 삭제
   - 해당 draft 작성 완료 시 `[결정]`, `[톤]` 삭제
   - 정리 결과를 사용자에게 보고 후 반영
4. **.claude/draft/ 스캔** — 실제 draft 파일 존재 여부와 series-hierarchy.md 대조
5. **브리핑 출력** — 채팅으로 현황 보고

## 규칙

- context_bridge 정리는 **사용자에게 보고 후** 반영 (삭제 전 확인)
- 그 외 출력은 **채팅 전용**
- context_bridge.md가 없으면 "이전 세션 기록 없음" 보고
