# 01 가상화 개념 / 01 가상화 정의와 목적

가상화가 무엇이며 왜 필요한지, 그리고 가상화가 해결하는 문제를 다룬다.

## What you will learn

- 가상화의 정의와 물리 자원의 추상화 원리
- 가상화 이전 "1 서버 = 1 애플리케이션" 모델의 문제점
- 서버 통합(Consolidation), 격리(Isolation), 탄력성(Elasticity)의 핵심 가치
- 가상화 이전 vs 이후의 차이
