# 01 가상화 개념 / 04 가상화와 클라우드

가상화 기술이 클라우드 컴퓨트의 기반으로 어떻게 이어지는지, 그리고 Container와 VM이 어떻게 공존하는지를 다룬다. Ch01 전체의 결론이다.

## What you will learn

- 클라우드 IaaS가 대규모로 자동화된 가상화라는 것
- AWS(Nitro/KVM), Azure(Hyper-V), GCP(KVM)가 가상화 기술을 어떻게 활용하는지
- 로컬 VirtualBox Lab과 클라우드 컴퓨트의 개념적 대응 관계
- Container와 VM의 격리 수준 차이와 프로덕션에서의 공존 구조
- MicroVM(Firecracker, Kata Containers)이 보여주는 가상화의 진화 방향
