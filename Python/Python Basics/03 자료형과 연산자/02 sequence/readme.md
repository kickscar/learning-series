# Sequence Types: list, tuple, range
이 디렉터리의 예제들은 Python의 **시퀀스(sequence) 타입**인 `list`, `tuple`, `range`의 기본 동작을 이해하기 위해 구성되어 있다.
각 시퀀스의 공통 인터페이스(인덱싱, 슬라이싱, 순회)와 가변/불변의 차이를 코드로 직접 확인하는 것이 목적이다.


## What These Examples Explain
이 예제들은 다음 개념을 설명한다.

- `list`의 가변성 및 요소 수정
- `tuple`의 불변성과 사용 목적
- `range`의 정수 구간 표현과 메모리 특성
- 인덱싱(`a[i]`)과 슬라이싱(`a[i:j]`) 규칙
- `*rest`를 이용한 기본 언패킹

## Examples
- `lab01` → 리스트 생성과 수정
- `lab02` → 튜플의 불변성
- `lab03` → range의 동작 방식
- `lab04` → 시퀀스의 인덱싱
- `lab05` → 시퀀스의 슬라이싱 I
- `lab06` → 시퀀스의 슬라이싱 II
- `lab07` → `*rest` 언패킹
