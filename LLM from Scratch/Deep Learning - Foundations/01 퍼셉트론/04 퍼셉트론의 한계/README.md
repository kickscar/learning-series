# 04 퍼셉트론의 한계 — XOR 문제

단층 퍼셉트론으로 XOR을 풀 수 없음을 수학적으로 증명하고 시각적으로 확인한다.

## 학습 내용

- XOR 진리표에서 도출되는 부등식 모순 — 단층 퍼셉트론으로 불가능한 이유
- 선형 분리 불가능(Linearly Inseparable)의 기하학적 의미
- Minsky & Papert(1969)와 AI 겨울의 역사적 맥락
- "단층의 한계 ≠ 신경망의 한계" — 다층으로의 복선
