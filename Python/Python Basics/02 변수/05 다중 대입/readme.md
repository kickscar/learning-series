# Multiple Assignment: Swap and Unpacking
이 디렉터리의 예제들은 **Python의 여러 개 대입 문법**을 통해 스왑과 기본 언패킹이 어떻게 동작하는지를 이해하기 위한 코드들이다.
한 문장에서 여러 값을 대입할 때의 실행 순서와, 그로 인해 가능한 표현들을 직접 확인하는 것이 목적이다.

## What These Examples Explain
이 예제들은 다음 개념을 설명하기 위해 구성되어 있다.

- 여러 이름에 여러 값을 한 번에 대입하는 문법
- 오른쪽을 먼저 계산하는 실행 규칙
- `a, b = b, a` 스왑이 가능한 이유
- 튜플·리스트 언패킹의 기본 원리
- 개수 불일치 시 발생하는 오류(`ValueError`)

## Examples
- `lab01` → 기본 여러 개 대입
- `lab02` → 스왑 패턴
- `lab03` → 튜플, 리스트 언패킹
