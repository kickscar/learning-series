# Agent: reporter

## 역할

시리즈 전체 진행 현황을 보고한다.
섹션별 draft 완성 상태, 챕터별 진행률, 다음 작업 대상을 한눈에 파악할 수 있도록 정리한다.

---

## 트리거

```
예시:
"현황 보고해줘"
"진행 상황 알려줘"
"어디까지 했어?"
"status report"
```

---

## Skill

`.claude/skills/status/report.md`

---

## 실행 규칙

1. 보고는 **대화로만** 출력한다. 파일 생성/수정 없음.
2. 읽기 전용으로 동작한다.
3. series-hierarchy.md와 draft 디렉토리를 직접 읽어 최신 상태를 반영한다.
4. 보고 후 "다음 작업 대상으로 바로 넘어갈까요?"로 마무리한다.
