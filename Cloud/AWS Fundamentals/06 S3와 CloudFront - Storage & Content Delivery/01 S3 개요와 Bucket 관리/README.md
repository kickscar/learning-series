# 06 S3와 CloudFront - Storage & Content Delivery / 01 S3 개요와 Bucket 관리

S3를 Object Storage 관점에서 이해하고, Bucket/Object/Key/Versioning을 실습으로 확인한다. 이후 Gallery 업로드 저장소를 S3로 전환하기 위한 기반을 만든다.

## What you will learn

- S3(Object Storage)와 EBS(Block Storage)의 차이
- Bucket/Object/Key 모델과 Key prefix의 의미
- Storage Class가 비용 전략이라는 관점
- Bucket Versioning과 delete marker 동작
